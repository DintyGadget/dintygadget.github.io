API-интерфейс Eluna Lua Engine©

API-интерфейс Eluna Lua Engine© позволяет вам добавить свой собственный код Lua, который будет выполняться при возникновении определенных событий (называемых «hook»)..

Добавьте новую внутриигровую команду, дайте жизнь существам с новым ИИ или добавьте кольцо огня вокруг игроков во время дуэли! Если «hook» существует, вы можете реализовать то, что задумали.

О Eluna

Eluna - это движок Lua для эмуляторов World of Warcraft. Данная версия Eluna поддерживает версию 9.0.5.

Чтобы скачать Eluna, скачайте эмулятор, с уже встроенной Eluna (версия эмулятора 9.0.5). Все что вам останется потом, просто скомпилировать проект, и начать кодить!

Классы

Aura

Постоянный эффект заклинания, который постоянно остается на юните после произнесения заклинания при помощи каста.
BattleGround

Содержит состояние поля битвы, например, Ущелье Песни войны, Нагорье Арати и т.д.
Corpse

Функции для игрока, после его смерти.
Creature

Неигровые управляемые юниты (т.е. NPC).
ElunaQuery

Результат запроса к базе данных.
GameObject

Наследует все методы от: Объект, WorldObject
Global

Эти функции можно использовать в любом месте в любое время, в том числе при запуске сервера.
Group

Наследует все методы от: нет
Guild

Наследует все методы от: нет
Item

Наследует все методы от: Объект
Map

Игровая карта, например, Азерот, Восточные Королевства, Огненные Недра и т.д.
Object

Базовый игровой объект (либо предмет или же WorldObject).
Player

Наследует все методы от: Объект, WorldObject, Юнит
Quest

Наследует все методы от: нет
Spell

Экземпляр заклинания, созданный при наложении заклинания юнитом.
Unit

Наследует все методы от: Объект, WorldObject
Vehicle

Наследует все методы от: нет
WorldObject

Наследует все методы от: Объект
WorldPacket

Пакет, используемый для передачи сообщений между сервером и клиентом.
