Класс Аура [-] [+]

Постоянный эффект заклинания, который постоянно остается на юните после произнесения заклинания при помощи каста.

Например, если вы накладываете заклинание с параметром "урон с течением времени", на цель накладывается аура которая наносит урон непрерывно.

Ауры вашего игрока отображаются в игре в виде серии значков слева от миникарты.

Наследует все методы от: нет

Методы

GetAuraId

Возвращает ID заклинания, которое применило ауру.
GetCaster

Возвращает юнита который использовал заклинание, которое применило ауру.
GetCasterGUID

Возвращает GUID юнита который использовал заклинание, которое применило ауру.
GetCasterLevel

Возвращает уровень юнита который использовал заклинание которое применило ауру.
GetDuration

Возвращает количество времени, оставшееся до истечения ауры.
GetMaxDuration

Возвращает количество времени, в течение которого эта аура действует при наложении.
GetOwner

Возвращает юнита на которого была наложена аура.
GetStackAmount

Возвращает количество раз, когда аура "стакалась".
Remove

Убрать ауру с юнита к которому она применяется.
SetDuration

Изменить количество времени, до истечения действия ауры.
SetMaxDuration

Изменить максимальное количество времени, до истечения действия ауры.
SetStackAmount

Изменить количество раз, когда аура "стакалась" на юните.
